Level 1

受注・出荷・売上

部材支給の小規模製造業に最適なスタートパッケージ

EDI連携による受注自動登録から、出荷ピッキング、納品書・請求書発行まで。受注から売上までのコアプロセスを一元管理します。

受注管理

  • EDI伝送データ（受注登録処理）
  • EDI変換テーブル管理
  • 受注伝票入力（量産品）
  • 受注伝票入力（未登録品）
  • 受注残高集計
  • 受注明細帳

出荷管理

  • ピッキングリスト
  • 出荷予定
  • 出荷入力（量産品/未登録品）
  • 出荷明細帳
  • 返品入力・明細帳
  • 検品/製品タグ入力・印刷

売上管理

  • 得意先別売上
  • 得意先別検収確認
  • 納品書作成
  • 請求書作成

Level 2

購買・外注・支払

自社で部材を調達・外注管理をする企業に

Level 1の機能に加え、部材の発注・仕入、外注発注管理を追加。自社で部材を購入する企業の業務効率化を支援します。

外注管理

  • 外注受入管理・明細
  • 部品注文書
  • 部品発注管理
  • 手書伝票入力・印刷
  • 外注注残・部品注残検索
  • 外注検収確定

購買管理

  • 購買伝票入力
  • 購入明細
  • 購買検収データ選択
  • 購買検収確定・取消

支払管理

  • 支払管理
  • 購買外注仮検収処理
  • 支払内訳

Level 3

生産計画・工程進捗・在庫管理

本格的な生産管理を実現するフル機能パッケージ

Level 1・2に加え、生産計画作成・MRP所要量展開・工程進捗管理・在庫管理が利用可能。一般的な生産管理に必要な機能がほぼ揃います。

生産計画・指図書

  • 推移表データ生成
  • 生産計画入力・推移表
  • 指図書採番（所要量展開）
  • 指図書発行情報・印刷
  • 特記事項入力
  • 注文書印刷

工程管理

  • 作業報告入力
  • 工程進捗管理・一覧
  • ロットトレース・一覧

在庫管理

  • 入庫処理（製品）
  • 素材・製品出庫処理
  • 出庫取消
  • 在庫メンテ
  • 棚卸管理・調査表
  • 製品未出庫リスト
